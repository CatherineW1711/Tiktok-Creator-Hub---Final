## Implmentation details/How to use: (ON MOBILE)
Open https://tiktokcreatorhub.netlify.app/ on Safari browser and grant permission to use camera. The home page
of the applucation appears. 

1. Click Start Dance Mode to enter the dance page
3. In Dance Mode:
  - Click Preview to start the dance preview
  - Click Stop Preview to stop it
  - On the top, choose the music you wish to dance for, and choose the dificulty level of poses that you want to learn
  - Click Tap to Record to start recording. While recording, the dance animation will play on the screen
  - Click the Record button again to stop recording
4. A Takes Management list will be displayed all the time below:
  - Click the blue replay icon (rotating arrow on the left) to replay the recorded take.
  - Click the red trash icon to delete a take (the take will be permanently removed)
5. Scroll down and click Publish at the bottom to publish your recoeded take (it shows the total number of takes to be published)
6. A message will appear: “Video published successfully.”
7. Click Return Home to go back to the main page

## Running the code on a COMPUTER:

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server on a localhost

  Make sure to install React in VS Code if you haven’t:
  
    npm install react react-dom
    npm install -D typescript @types/react @types/react-dom
