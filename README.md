Clone the project and open in VScode

## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  

 Code is generated by Figma Make AI

## Implmentation details/How to use: 
Once you follow the setup steps above and run the code, a local website will automatically open in your browser (localhost).

1. On the Home Page, click on the Username and Password fields and type your inputs using your physical keyboard (not by clicking on-screen keys)
2. Click Start Dance Mode to enter the dance page
3. In Dance Mode:
  - Click Preview to start the dance preview
  - Click Stop Preview to stop it
  - Click Tap to Record to start recording. While recording, the dance animation will play on the screen
  - Click the Record button again to stop recording
4. A Takes Management list will appear:
  - Click the blue replay icon (rotating arrow on the left) to replay
  - Click the red trash icon to delete a take (the take will be permanently removed)
5. Scroll down and click Publish at the bottom (it shows the total number of takes to be published)
6. A message will appear: “Video published successfully.”
7. Click Return Home to go back to the main page
